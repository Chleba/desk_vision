# desk_vision
Local AI Image search

## Installation

- Install [Ollama](https://ollama.com/) on your PC that will be running Ollama models (need some good graphic card or some Apple sillicon mac)
- pull vision model into a Ollama ([https://ollama.com/search?c=vision](https://ollama.com/search?c=vision). Open Power Shell/terminal and write - `ollama pull llava` or `ollama pull llava-llama3`
